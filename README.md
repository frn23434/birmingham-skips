# Wolverhampton Skips

A full-stack TypeScript app using Vite (React) on the frontend and Express on the backend.

## Requirements

- Node.js 20+
- npm 10+

## Local development

```bash
npm install
npm run dev
```

The server runs on `http://localhost:5000` by default.

## Build and start

```bash
npm run build
npm run start
```

## Scripts

- `npm run dev` - start dev server
- `npm run check` - TypeScript type check
- `npm run build` - production build
- `npm run start` - run production server

## GitHub

This repository includes a GitHub Actions workflow at `.github/workflows/ci.yml` that runs type checks and builds on pushes and pull requests.
